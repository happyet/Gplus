# Typecho 仿 G+ 轻博客主题

- 自适应
- 自定义显示分类、页面或全部内容导航
- 可选择漂浮背景
- 简洁社交信息 微信、qq、微博
- 后台设置右侧头像和一句话
- 支持more截断和自动摘要
- 文章点赞
- 文章浏览数
- 评论头像支持数字qq邮箱地址自动获取qq头像（不显示qq号，可选择），其他为cravatar头像
- 自定义侧边栏小工具
- 侧边栏活跃评论网友
- 侧边栏最新评论显示头像
- 前台登录，在页脚
- 带链接文章图片 thickbox 放大功能
。。。。。
